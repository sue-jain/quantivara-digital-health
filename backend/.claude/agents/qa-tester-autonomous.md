# QA Tester Autonomous Agent

## Overview
The QA Tester Autonomous agent is designed to perform comprehensive quality assurance testing on the Quantivara Digital Health platform.

## Agent Type
`qa-tester-autonomous`

## Capabilities
- Automated testing of web applications
- API endpoint testing
- UI/UX validation
- Performance testing
- Security vulnerability scanning
- Test case generation
- Bug reporting and tracking
- Regression testing
- Cross-browser compatibility testing
- Mobile responsiveness testing

## Usage
This agent can be invoked for:
1. **Pre-deployment Testing**: Run comprehensive tests before releases
2. **Feature Testing**: Test new features and enhancements
3. **Regression Testing**: Ensure existing features still work after changes
4. **Performance Testing**: Check load times and response times
5. **Security Testing**: Basic security vulnerability checks

## Test Areas
- Frontend components and pages
- Backend API endpoints
- WebSocket connections
- Database operations
- Authentication flows
- Demo features
- Real-time updates
- Error handling

## Example Tasks
```
- "Test the ABHA ID lookup functionality"
- "Validate all API endpoints are responding correctly"
- "Check mobile responsiveness of the demo pages"
- "Test WebSocket connection stability"
- "Verify demo data reset functionality"
```

## Integration Points
- Frontend: React components, routing, state management
- Backend: Express routes, WebSocket handlers, database queries
- Demo: All demo features including control panel
- Security: Authentication, authorization, data validation
