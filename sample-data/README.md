# Sample Data Directory

This directory contains sample medical data for prototype development and demo purposes.

## Directory Structure

```
sample-data/
├── prescriptions/          # Sample prescription forms and images
├── lab-reports/           # Sample blood test results and lab reports
├── patient-timeline/      # Connected patient journey examples
└── demo-scenarios/        # Complete demo workflows
```

## Data Sources

Based on internet research, I found these resources for sample medical data:

### Prescription Templates
- Template.net: Professional prescription templates in PDF format
- Jotform: Customizable prescription PDF templates
- Carepatron: Free prescription templates for medical practices
- PrescriptionTemplates.com: MS Word format templates

### Lab Report Formats
- PDFFiller: CBC report PDF templates
- My Lab India: Various blood test report formats
- Carepatron: MCH Blood Test Report templates
- LabSmart LIS: Comprehensive pathology report formats

## Data Types Needed

### Prescriptions
- Standard prescription forms with patient info, medications, dosages
- Lab order prescriptions (CBC, metabolic panel, etc.)
- Various doctor handwriting styles
- Different medication types and combinations

### Lab Reports
- Complete Blood Count (CBC) results
- Basic Metabolic Panel (BMP) results
- Comprehensive Metabolic Panel (CMP) results
- Normal and abnormal values for demo purposes
- Different lab format styles

### Patient Timeline Data
- Connected care journeys showing:
  - Initial doctor visit → Lab order → Results → Follow-up
  - Multiple visits over time showing trends
  - Different conditions and treatments

## User Data Integration

**Important**: User will upload real prescription and lab data from family members for reference. These will be:
- Anonymized for privacy
- Used as realistic examples for AI training
- Incorporated into demo scenarios

## Usage in Prototype

This sample data will be used for:
1. Training AI models for OCR and data extraction
2. Creating realistic demo scenarios
3. Testing the integration between doctor-lab-hospital workflows
4. Validating AI accuracy on real medical document formats
